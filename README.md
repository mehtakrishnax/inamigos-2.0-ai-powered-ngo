# inamigos-2.0-ai-powered-ngo
**InAmigos 2.0 is an AI-powered NGO platform designed to enhance social impact through intelligent technology. The platform features AI-driven volunteer matching, smart donation recommendations, impact analytics, an AI assistant, and community engagement tools. Built with a modern and responsive design, it aims to connect volunteers, donors, and communities while improving accessibility, transparency, and project management. This project showcases how Artificial Intelligence can help NGOs create greater impact and deliver meaningful change more efficiently.
